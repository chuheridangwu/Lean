# Sorry,sir...! 
课文内容:
```markdown
My coat and my umbrella, Here is my ticket      // 请把我的大衣和伞拿给我，这是我(寄存东西)的牌子
Thank you,sir。 Number five   // 谢谢，先生， 是 5 号。
Here's your umbrella and your coat         // 这是您的伞和大衣
This is not my umbrella   // 这不是我的伞
Sorry,sir.  Is this your umbrella?     // 对不起，先生。这把伞是您的吗？
NO, it isn't.       // 不，不是？
Is this it ?        // 这把是吗？
Yes,it is. Thank you very much.     //是，是这把。非常感谢。
```
